
## Why should I use it?

There have been some improvements in performance, stable diagnostic reports, [V8 using version 8.1](https://v8.dev/blog/v8-release-81) with new cool features like [nullish coalescing](https://wiki.developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Operators/Nullish_Coalescing_Operator), [modules](https://nodejs.org/docs/latest-v14.x/api/esm.html#esm_modules_ecmascript_modules), [displayNames](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/DisplayNames), etc. In addition is the new LTS.
